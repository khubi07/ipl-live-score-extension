# 🏏 IPL Live Voice Commentary Extension

An AI-powered Chrome extension that provides live IPL score updates with voice commentary.

## ✨ Features

- 📡 Live IPL score updates
- 🎙 Voice commentary using SpeechSynthesis API
- 🌙 Modern dark UI
- ⚡ Lightweight Chrome extension
- 🔴 Real-time match information

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- Chrome Extension Manifest V3
- Browser SpeechSynthesis API

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/khubi07/ipl-live-score-extension.git
```

### 2. Open Chrome Extensions

```txt
chrome://extensions
```

### 3. Enable Developer Mode

Turn ON developer mode from top-right corner.

### 4. Load Extension

Click:

```txt
Load unpacked
```

Then select the project folder.

## 🎤 Voice Commentary

This extension uses the browser's built-in:

- SpeechSynthesis API
- Text-to-Speech engine

to generate spoken IPL commentary.
